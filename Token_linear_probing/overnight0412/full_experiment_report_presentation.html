Q1. 위치 정보가 특정 subspace에 있는가?
No — 전체 D=1152에 분산 인코딩. PCA 50개(분산 82%)로는 R²(x)=0.06에 불과.
Q2. 시각적 복잡도가 direction encoding을 약화시키는가?
Yes — 특히 배경 복잡도가 주범. 실제 배경 추가 시 direction-strong dim 665개→99개(85% 감소).
Q3. LLM의 last token이 vision 정보를 가져가는가?
Yes — Layer 1부터 즉시 흡수. Layer 4에서 peak (direction 57%, pos R²=0.85). 끝까지 chance 이상.
Phase 1 — Position Subspace 분석
Q: 위치 정보가 특정 low-rank subspace에 집중되어 있는가?

k = {1, 2, 3, 5, 10, 20, 50, 100, 200, 500, 1000} 각각 5-fold CV R²

Phase 2 — 시각적 복잡도 Gradient
Q: 물체/배경이 복잡해지면 direction encoding이 약해지는가?
실험 방법
4개 조건 × 200 videos (testbed)
shape_color (도형 + 단색) / obj_color (실제물체 + 단색) / shape_place (도형 + 실제배경) / obj_place (실제물체 + 실제배경)

Phase 3 — LLM 내부 Hidden State Probing

Last token hidden state 추출
LLaVA-Video-7B-Qwen2 (fp16, ~14GB) 로드

Phase 4 — 학습된 모델 Dim 구조 비교 (Rigorous)
Q: Fine-tuning이 projector의 direction dim 구조를 어떻게 바꾸는가?
실험 방법
Per-patch projection → mean-pool (정확한 계산)
4개 모델의 projector weight 로드: vanilla / baseline-v5 / delta_direct-v5 / channel_gate-v3
동일한 pre_mean features(8000, 8, 729, 1152)에 각 projector를 patch별로 적용 후 mean-pool
post = mean(projector(patch_1), ..., projector(patch_729))
→ post_mean (8000, 8, 3584) 계산, 이전 approximation 오류 수정

측정: Delta direction acc / Position R² / Object acc / Direction F-stat / Dim overlap
Direction F-stat (per-dim)
delta = feat[:, -1, :] − feat[:, 0, :] (마지막−첫 프레임)
각 dim d에 대해 4방향 그룹 간 one-way ANOVA F-통계량 계산
Top-50 dim 선정 → 모델 간 집합 교집합(overlap) 계산

핵심 해석:
Baseline: direction explicit signal 없이 projector tuning만으론 dim 구조 거의 안 바뀜 (46/50 그대로).
Delta-direct/Channel-gate: vanilla가 쓰던 direction dim 32~33개를 강화(F-stat +40%)하면서 추가로 새 dim도 개척. "갈아엎는 것"이 아니라 "기존을 증폭 + 신규 개척".
⚡ 이전 approximation 실험과 결론 역전:
근사: "delta_direct가 vanilla dim 버리고 완전히 새 dim 만든다" (14/50 overlap)
정확: "vanilla dim 유지/강화하면서 일부 추가" (21/50 overlap, 강화 32/50)
Phase 5b — 실제 영상 (SSv2, KTH)
Q: 합성 데이터에서 발견한 direction encoding이 실제 영상에서도 작동하는가?
데이터 및 실험 방법
SSv2 (Something-Something v2)
left/right pushing 각 250 videos → 총 500개
2-way classification (chance = 50%)
SigLIP + Vanilla projector로 on-the-fly feature 추출 → 8 frames uniform sample
KTH Action Dataset
left/right walking/jogging: left 447, right 452 → 총 899개
동일하게 2-way, SigLIP + projector feature 추출

SSv2: single frame에서는 방향을 알 수 없음(0.452 ≈ chance 50%). push라는 동작이 발생해야 방향이 드러남 → delta가 핵심.

KTH: single frame만으로 73.3%. 사람이 걷는 방향이 신체 자세/위치에서 이미 보임.

Post-proj에서 F>5 dim 폭증 (SSv2: 49→441, KTH: 390→1400): projector가 실제 영상에서도 direction 신호를 여러 dim에 강하게 확산시킴.
합성 데이터(R2R_real_color)에서 발견한 패턴이 실제 영상에서도 동일하게 작동함을 확인. 외부 타당성 확보.
논문 연결 — 전체 실험의 시사점
① Vision encoder는 방향 정보를 이미 갖고 있다
Delta direction R² 88~92%, F-stat top-50 집중. 정보는 있다.
② LLM lm_head가 이를 방향 토큰으로 못 꺼낸다
Layer 28에서도 direction 47% — 정보는 살아있는데 디코딩 실패. 병목은 lm_head.
③ Projector tuning으로 신호를 증폭하면 LLM이 더 잘 읽는다
Delta-direct/Channel-gate: F-stat +40%, direction acc +4%p. Projector가 핵심 레버.
④ 배경 노이즈 제거가 추가 개선 여지
실제 배경 시 direction-strong dim 85% 감소. Background suppression이 실질적 도움 될 것.
